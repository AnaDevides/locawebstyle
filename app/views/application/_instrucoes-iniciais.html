Informações iniciais

Além de ler aqui, você pode ler algumas instruções importantes no README.MD.

Este é um guia de código para facilitar a construção de elementos visuais de layout dos serviços da Locaweb. Você encontrará dois tipos de códigos aqui: HTML e CSS. Não encontrará, ainda, nada de Javascript, porque a maioria das funções são baseadas em frameworks como Bootstrap e Modernizr.

Todos os códigos de CSS são importados automaticamente assim que você linka o arquivo application.css. Logo o código CSS que você encontrar aqui é puramente pra sua consulta. Não modifique para nada. Este guia será atualizado com frequencia ao longo do tempo, por isso suas modificações serão perdidas assim que atualizarmos o código para novas versões.
