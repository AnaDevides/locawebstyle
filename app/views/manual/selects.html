Selects customizáveis

Customização de select é normal em alguns projetos. Na Locaweb utilizamos o Chosen para customizar nossos selects. Ele é o plugin mais completo e acessível que encontramos.

Para utilizá-lo é simples, basta usar a classe customSelect no select que você quer customizar e pronto.

O atributo data-placeholder mostra uma frase no select para indicar alguma instrução para o usuário.

Para melhor usabilidade, quando houverem mais de 20 itens no select, nós mostramos automaticamente um input de search no select. Assim facilitamos a procura de opções.



<%= content_for_clippy do %> <% end %>

Múltiplas escolhas

Para selects com múltiplas escolhas o Chose modifica de forma que o visual fica com o abaixo:


<%= clippy(yield(:code)) %>
