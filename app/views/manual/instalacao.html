Instruções de instalação

Primeiro passo: Adicione as tags abaixo para adicionar o pacote de CSS e Javascript.

<link rel="stylesheet" type="text/css" href="<%= File.join(root_url, path_to_asset('application.css')) %>">
<script type="text/javascript" src="<%= File.join(root_url, path_to_asset('application.js')) %>"></script>
