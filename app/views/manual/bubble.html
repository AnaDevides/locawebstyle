Name type default description
animation boolean true Aplica uma transição de fade
placement string|function 'right' Onde o popover ficará posicionado - top | bottom | left | right
selector string false Se um seletor é definido, os objetos do popover será delegado para alvos específicos
trigger string 'hover' Como o popover é ativado - hover | focus | manual
title string | function '' Título padrão do popover se o atributo title não existir
content string | function '' Conteúdo padrão se o data-content não existir
delay number | object 0

Delay para mostrar e esconder o popover (ms) - não aplicável para ativação manual.

Se um número é fornecido, o delay é aplicado no hide e show.

Estrutura do objeto é: delay: { show: 500, hide: 100 }
