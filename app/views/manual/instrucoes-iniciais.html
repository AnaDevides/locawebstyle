Informações iniciais

O Locaweb Style são os padrões de interface utilizados nos produtos da Locaweb. Ele explora um simples e flexível HTML, CSS e Javascript para as necessidades comuns na manutenção e criação dos nossos serviços.

Este projeto foi criado para manter uniforme a identidade de todas as interfaces dos produtos da Locaweb. Outro Objetivo importante para tal iniciativa é a uniformização do nosso código, melhorando a perfomance de produção, facilitando o aprendizado de novos integrantes e agilizando a implementação de novas ideias.

Com essa estrutura, integramos de forma consistente as necessidades de interface e programação, trabalhando em um código client-side robusto e escalável. Assim mantemos a fidelidade do layout sob um código limpo e enxuto, que se adequa às necessidades dos programadores.

O que o Locaweb Style utiliza?

Nós utilizamos alguns plugins ja conhecidos de todos os devs e que não precisam ser inseridos novamente no projeto. Se precisar de algum plugin ou um novo adendo às funcionalidades, fale conosco.

Bootstrap

Os estilos e comportamentos foram baseados no Bootstrap, framework criado pelo Mark Ott e Jacob Thornton para agilizar a criação de interface do Twitter. Customizamos o framework para termos apenas o que for necessário.

Se você não conhece, visite o github do projeto e conheça já!

Curiosidade – Dentro do Bootstrap eles usam o Normalize, que é um CSS Reset bastante completo. Ele padroniza e compatibiliza alguns problemas de textos e valores padrões em elementos do HTML. O Normalize foi criado pelo Nicolas Gallagher and Jonathan Neal. Dê uma olhada no projeto

JQuery

O JQuery é o framework mais utilizado para controle e manipulação de comportamentos de elementos do HTML. Acho que ele dispensa apresentações. Link do projeto

Modernizr

A Modernizr é um script de detecção de compatibilidade e suporte dos browsers ao CSS3 e algumas features do HTML5. Ele nos avisa se o browser suporta ou não as novidades do W3C, possibilitando a criação de fallbacks para browsers incompatíveis. Assim fica fácil saber quais browsers suportam ou não SVG, canvas, RGBA, drop shadow, gradiente e tudo o mais.

Ele foi criado por desenvolvedores renomados como Paul Irish e Faruk Ates. Site do projeto

Chosen

Chosen é um plugin para customização de selects (ou como muitos chamam "combo box") muito útil e flexível. Ele é o melhor plugin para executar essa tarefa por que ele mantém a acessibilidade. Você consegue navegar via teclado e etc. Para ver como utilizamos, visite esse link.

Masked Input

Masked Input é um plugin para a criação de máscaras para formulários. É muito simples de utilizar. Para ver seu funcionamento, clique aqui.
