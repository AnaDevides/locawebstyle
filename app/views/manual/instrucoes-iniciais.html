Informações iniciais

O Locaweb Style Guide explora um simples e flexível HTML, CSS e Javascript para as necessidades comuns na manutenção e criação dos serviços da Locaweb. Isso é uma ferramenta para a criação e manutenção de maneira simples e rápida.

Este projeto foi criado para que a criação de novos serviços e a manutenção do legado fosse simples e rápida. Outro Objetivo importante para tal iniciativa é uniformizar o código de toda a Locaweb, melhorando a perfomance de produção, facilitando o aprendizado de novos integrantes e agilizando a implementação de novas ideias.

O que o Locastyle utiliza?

Já existem alguns plugins que nós utilizamos aqui e que você não precisa importar novamente em seu projeto, abaixo segue a descrição. Se precisar de algum plugin ou um novo adendo às funcionalidades, fale conosco.

Bootstrap

Os estilos e comportamentos foram baseados no Bootstrap, framework criado pelo Mark Ott e Jacob Thornton para agilizar a criação de interface do Twitter. Customizamos o framework para termos apenas o que for necessário.

Se você não conhece, visite o github do projeto e conheça já!

Curiosidade – Dentro do Bootstrap eles usam o Normalize, que é um CSS Reset bastante completo. Ele padroniza e compatibiliza alguns problemas de textos e valores padrões em elementos do HTML. O Normalize foi criado pelo Nicolas Gallagher and Jonathan Neal Dê uma olhada no projeto

JQuery

O JQuery é o framework mais utilizado para controle e manipulação de comportamentos de elementos do HTML. Acho que ele dispensa apresentações. Link do projeto

Modernizr

O Modernizr é um script de detectação de compatibilidade do CSS3. Ele nos avisa se o browser suporta ou não suporta as novas propriedades do CSS, nos dando a possibilidade de criar fallbacks para os browsers incompatíveis. Assim fica muito fácil saber qual browser suporta ou não SVG, Canvas, RGBA, drop shadow, gradiente e tudo o mais.

Ele foi criado por desenvolvedores renomados como Paul Irish e Faruk Ates. Site do projeto

eCSStender

eCSStender é um pluginzinho lindo que faz com que os browsers, quer dizer, browsers não, IEs, entendam coisas maravilhosas que o CSS faz.

Leia mais sobre o eCSStender

JQuery UI

JQuery UI é um framework baseado em comportamento que facilita a criação e a manutenção de alguns comportamentos da página, como Datepicker, animações e muitas outras necessidades.

Também customizamos o download do JQuery UI para usarmos apenas o necessário. Se você precisar utilizar algo a mais, nos de um toque, assim implementaremos para todos.
