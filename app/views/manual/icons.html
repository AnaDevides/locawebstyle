Elementos com ícones

O Bootstrap já tem um esquema de inserção de ícones em elementos. No Locastyle modificamos esse funcionamento para que não fosse necessário adicionar um elemento a mais.

Nós também temos a necessidade de criar novos ícones de diferentes cores para os serviços. O ponto importante foi fazer com que você possa usar nossos ícones e usar os ícones do Bootstrap. Há um prefixo nas classes: icon- para os ícones do Bootstrap e ico- para os ícones da Locaweb.
Precisamos de duas classes (por enquanto) porque usamos dois sprites diferentes.

O funcionamento é simples. As classes podem ser inseridas em qualquer elemento, mas é tem um efeito muito melhor em elementos de linha, como links, span, strong, i, em, etc... Isso quer dizer que talvez, quem sabe, a classe vai funcionar em uma TD, mas o efeito pode não ser o esperado.

Para adicionar seu ícone, insira uma das classes abaixo no elemento escolhido.
