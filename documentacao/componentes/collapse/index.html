Atenção

Nos exemplos usamos o elemento a com a classe .ls-collapse-header, mas qualquer elemento, como um DIV por exemplo, pode receber essa classe.

Parâmetros adicionais

Nome Tipo Descrição
ls-collapse-open-always classe mantém um collapse sempre aberto
ls-collapse-open classe inicia um collapse aberto

Accordeon

Accordeon é um conjunto de collapses agrupados, onde apenas um deles permanece aberto.

Um accordeon é um conjunto de collapses, como mostrados acima, com um elemento que os agrupa, com a classe .ls-collapse-group.

Os collapses agrupados em um accordeon aceitam as mesmas configurações que um collapse normal.

Abrir com ação externa

Para usar um botão ou link para controlar um collapse utilize o atributo data-toggle-collapse com o valor do id do collapse.

Para abrir/fechar um collapse usando javascript, use:

1
